# OzTaxTracker 🇦🇺

A backend API for tracking work-related expenses and estimating potential tax returns in Australia.

## Tech Stack
* Java 21
* Spring Boot 3.x
* Spring Data JPA
* H2 Database (for development)
* Lombok & Swagger UI

## Why this project?
To solve the complexity of managing tax-deductible expenses for professionals in Australia.
